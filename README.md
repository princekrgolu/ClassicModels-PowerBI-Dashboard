# 📊 Classic Models Sales Performance Dashboard

## 📌 Overview

This project is an interactive **Business Intelligence dashboard** developed using **Power BI**, **MySQL**, and **DAX** to analyze sales performance using the Classic Models sample database.

The dashboard enables users to monitor key business metrics, identify sales trends, compare year-over-year performance, and analyze revenue across product lines, customers, and countries through interactive visualizations.

---

## 🎯 Project Objectives

- Design a professional Business Intelligence dashboard using Power BI.
- Build a star-schema data model from the Classic Models MySQL database.
- Implement DAX measures and Time Intelligence calculations.
- Provide interactive sales insights using KPIs, charts, and slicers.
- Demonstrate industry-standard BI development practices for portfolio and interview preparation.
  
---

## 📂 Dataset

- **Source:** Classic Models Sample Database (MySQL)
- **Domain:** Retail Sales
- **Tables Used:**
  - Customers
  - Orders
  - Order Details
  - Products
  - Product Lines
  - Payments
